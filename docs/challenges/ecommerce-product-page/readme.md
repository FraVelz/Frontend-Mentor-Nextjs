# Frontend Mentor — E-commerce product page

Solución al reto [E-commerce product page](https://www.frontendmentor.io/challenges/ecommerce-product-page-UPsZ9MJp6) integrada en este monorepo Next.js.

## El reto

El usuario puede:

- Ver un layout adecuado según el ancho de pantalla (mobile / desktop de referencia 1440px).
- Ver estados hover en los elementos interactivos.
- Abrir la galería en lightbox desde la imagen principal.
- Cambiar la imagen principal con las miniaturas.
- Añadir productos al carrito, abrir el carrito y eliminar líneas.
- Usar el menú de navegación en vista móvil.

## Implementación

- [Next.js](https://nextjs.org/) (App Router), React y TypeScript.
- Estilos con **Tailwind CSS**; tipografía **Kumbh Sans** vía `next/font`.
- Estado del carrito con contexto React; lightbox y modal de carrito como capas sobre la página.

Detalle visual (colores, tipografía): [style-guide.md](./style-guide.md).
